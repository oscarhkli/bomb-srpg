# Bomb Tactics
Turn-based Strategy Game

The first version is a simplified command-line interface (CLI) game. A web server version is currently in progress.

## Prerequisites

To build and run this project, you need to install **Go** on your system. 

---

## Usage

Build the project and run the CLI:

```bash
make build
./bin/srpg-cli
```
